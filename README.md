# fastapi-boilerplate
In order to create an template to FastAPI projects. :rocket:

## Development Requirements
- Python3
- Pip
- Docker (to deployment)

## Installation
> `make install`

## Runnning Localhost
> `make run`

## Deploy app
> `make deploy`

## Running Tests
> `make test`

## Access Documentation
> <http://localhost:8080/docs>
